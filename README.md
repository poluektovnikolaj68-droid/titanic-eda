# Titanic EDA

Исследовательский анализ данных пассажиров Титаника.

# Данные
Kaggle Titanic Dataset (train.csv) — 891 пассажир, 12 признаков.

# Что исследовал
- Базовая статистика и пропуски
- Фильтрация по классу, полу, возрасту (loc/iloc, булевы маски)
- Выживаемость по группам

# Главные находки
- Выжило 38,43% пассажиров
- Женщины выживали в 3 раза чаще мужчин
- Самый дорогой билет — 512 долларов, все топ-10 — 1й класс
- Только 7,8% мужчин 3-го класса выжили
